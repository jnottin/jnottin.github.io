My Projects

Medi-Share

Medi-Share is a web application that shows average prices of procedures for uninsured patients at hospitals near them. It is extremely hard to know the cost of a procedure before having it, through Medi-Share I wanted to bring more transparency to the medical community.

Technologies Used:

  • React
  • React Google Maps
  • React Google Autocomplete
  • Express
  • MongoDB

Conjugate This

Conjugate This is a language learning web application that allows respondents to study a set list of verbs. They can practice how to use those verbs in present, past and future tense as well as in three different languages: Spanish, Italian and Portuguese.

Technologies Used:

  • HTML
  • CSS
  • JavaScript

Trail To The Best

Trail To The Best is a web application inspired because I love going to the movies, especially with friends. It's very callenging though, to agree on which movie to see. This app allows a group of friends to upload movie trailers and then have any of their friends watch and rate those trailers to figure out which is the best fit for everyone.

Technologies Used:

  • Bootstrap
  • Handlebars
  • Express
  • MongoDB

RoomKind

RoomKind is a team developed web application that allows everyday people and shelters to list available beds for those in need, all for free! Shelters can list any additional unused beds they have at their centers and homeowners can also list their property, like Airbnb!

Technologies Used:

  • React
  • Express
  • MongoDB
